About

A simple pastebin-like website, where users may anonymously upload plaintext and file attachments up to 35 MiB in total, and share the shortened links with others.

Star it on GitHub

Copycat was made by Luke Wilson in one week, with about 18 hours in development.

The application utilizes common Amazon Web Services:

  1. RDS PostgreSQL - Relational Database Service.
  2. S3 - Simple Storage Service for storing file attachments.
  3. EC2 - Elastic Cloud Compute runs the webserver written in Go.
  4. Elastic Beanstalk - For managing the EC2 clusters and load balancer.
  5. Load Balancer - Distributes network traffic between EC2 clusters for scaling.
  6. VPC - Virtual Private Cloud for containerizing network services and restricting access.
  7. IAM - Access management and service permissions.

In addition, the application uses:

  1. Go programming language for backend.
  2. JavaScript, HTML, and CSS for frontend.
  3. Gin web framework for routing and network requests.
  4. GitHub and Git for source control.
  5. AWS SDK for Go to access the S3 bucket.
  6. SHA-1 hash algorithm for generating unique object IDs.

Motivation

To improve my web development skills, and to provide proof that I am capable of full-stack development, DevOps, security, and general software engineering.
